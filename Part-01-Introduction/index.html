Full Stack Web Development in Java - Part 2

Introduction

Welcome to our course on Full Stack Web Development using Java. In this session, we’ll understand what it means to be a full stack developer, why Java is a powerful choice, and we’ll even set up our first simple project.

What is Full Stack Development?

A full stack developer is someone who can work on:

  • Frontend – the part users see (HTML, CSS, JavaScript)
  • Backend – the part users don’t see (Java, database logic)
  • Database – storing and managing data (MySQL, MongoDB)

Analogy: Think of building a website like running a restaurant:

  • Frontend is the menu and service
  • Backend is the kitchen
  • Database is the storage of ingredients

Java for Full Stack Development

1. Front-end Technologies

  • HTML, CSS, JavaScript
  • Modern frameworks: Angular, React, Vue
  • Node.js and NPM for JavaScript runtime and package management

2. Back-end Technologies

  • Core Java – foundation for Java EE
  • Spring Framework & Spring Boot – simplify enterprise Java development
  • Hibernate – ORM for database interaction
  • Spring Security – authentication & authorization
  • Microservices – using Spring Boot & Spring Cloud

3. Databases

  • Relational: MySQL, PostgreSQL, Oracle
  • NoSQL: MongoDB

4. Tools

  • Apache Maven, Gradle, Git, Jenkins
  • Eclipse, IntelliJ IDEA, Docker, JIRA
  • Apache Ant, Java Decompiler

Live Demo: Spring Boot Setup

  1. Visit https://start.spring.io
  2. Select Maven, Java, Spring Boot 3.x
  3. Add dependencies: Spring Web, Spring Data JPA, MySQL Driver
  4. Generate and extract ZIP file
  5. Open in IntelliJ or Eclipse
  6. Create a REST Controller: 
    
@RestController
public class HelloController {
    @GetMapping("/hello")
    public String sayHello() {
        return "Hello from Spring Boot!";
    }
}
  7. Run and access http://localhost:8080/hello
